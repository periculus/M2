M2 Grammar Standalone Test

Open DevTools and inspect the spans to see if they have the correct CSS classes:

  • Keywords (if, then, else) should have class="cm-keyword"
  • Types (QQ, ZZ, RR) should have class="cm-typeName"
  • Functions (ideal, gb, res) should have class="cm-functionName"
  • Comments should have class="cm-lineComment" or "cm-blockComment"
  • Strings should have class="cm-string"
  • Numbers should have class="cm-number"
