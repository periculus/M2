The Problem

The parser works perfectly and identifies all tokens correctly. However:

  1. The Lezer highlight integration (`propSources: [m2Highlighting]`) is not working
  2. JupyterLab's theme CSS variables are overriding everything to green
  3. The webpack bundling process may be stripping the highlight mappings
  4. Even with CSS injection, the CodeMirror classes aren't being applied to tokens

This is why everything appears green in JupyterLab despite the parser working correctly.
